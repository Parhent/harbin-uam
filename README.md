# 哈尔滨空中交通数字孪生系统

## 项目概述
- 一个展示城市空中交通（UAM）调度的轻量级Web演示
- 包含：哈尔滨3D风格地图 + eVTOL航线模拟 + 路径规划可视化

## 技术栈
- 前端：原生HTML + JavaScript
- 地图：Leaflet（开源地图库）
- 动画：requestAnimationFrame

## 文件结构
- `index.html` - 主页面（地图+控制面板）
- `style.css` - 样式
- `app.js` - 核心逻辑（路径规划、调度、渲染）
- `data/waypoints.json` - 航点数据

## 功能
1. 地图展示 - 哈尔滨市区3D风格
2. 标记点 - 起点/终点/途经点
3. eVTOL模拟 - 动画展示飞行
4. 路径可视化 - 绘制航线
5. 高度层展示 - 不同颜色表示不同高度
6. 多机调度模拟 - 展示防冲突逻辑

## 坐标数据（暂定）
